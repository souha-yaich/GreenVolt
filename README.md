# DESCRIPTION OF MOBILE APPLICATION
The Green Volt mobile application is a core component of the solar energy system, designed to offer users a seamless and intuitive experience in managing their energy usage. With a user-friendly interface and a focus on real-time insights, the application bridges the gap between complex system operations and everyday usability. One of its standout features is real-time monitoring of the battery’s state of charge (SoC), giving users instant visibility into their stored energy levels. The application also supports a rule-based energy management system, which intelligently distributes electricity across four connected zones according to user-defined priorities and battery levels, ensuring optimal energy allocation during limited supply periods. Users can easily configure each zone, update component information, and assign custom priorities. Based on this data, the application automatically calculates the energy consumption of each area, offering a clear view of overall demand. Additionally, the autonomy hours estimation feature dynamically projects how long the system can continue to operate based on current consumption and battery charge, whether or not the rule-based system is active. Green Volt also provides a solar production estimate, helping users anticipate daily energy generation and adjust their usage accordingly. To enhance connectivity and reliability, the application utilizes Wi-Fi technology, allowing for a smoother setup and stable data access without complex hardware requirements. To further empower users, a real-time email notification is sent whenever the battery charge drops below 30%, enabling timely action and greater system awareness. Together, these features make Green Volt a smart, proactive solution for managing residential solar energy systems with confidence and efficiency.
